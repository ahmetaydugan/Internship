# Artificial Intelligence and Operational Processes Internship Log

This repository is created to regularly track and report the artificial intelligence fundamentals I acquire, practical coding tasks, and daily operational processes throughout my Turkcell internship program.

## Daily Work Rhythm

The standard work schedule followed every day during the internship is as follows:

*   **Academic Block:** Readings and video tutorials planned to establish the foundations of artificial intelligence.
*   **Practical Coding Task:** Translating the theoretical knowledge learned in the academic block into code and implementing algorithms.
*   **Daily Team Operations:** Daily operational tasks and team support duties communicated via email.
*   **Friday Presentations:** A 15-minute brief presentation every Friday to share the week's learnings and completed projects.

---

## Repository Structure

To make tracking easier, the repository is structured on a weekly basis. Each week's folder contains the codes and academic notes for the work done on those days.

* `Week_01/`
  * `Day_01/` (e.g., Article notes, Python scripts)
  * `Day_02/`
  * `Presentations/` (15-minute presentation files delivered on Fridays in .pdf or .pptx format)
